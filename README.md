# In-Principle Replication: Earnings Management and Investor Protection

This repository contains the data and Jupyter Notebook analysis for an in-principle replication of:

Leuz, C., Nanda, D., & Wysocki, P. D. (2003). Earnings management and investor protection: An international comparison. Journal of Financial Economics, 69(3), 505â€“527.

The replication focuses on one earnings management measure (EM1) and one proxy for investor protection (the shareholder rights index) using contemporary firm-level data.

data.csv : Compustat Global - Fundamentals Annual data for years 2016-2025. Firm-level dataset used for the analysis (operating income, operating cash flow, fiscal year, country codes, etc.)

shareholder_rights_data.csv : Data for country-level shareholder rights index from World Bank Doing Business Database for year 2019.
